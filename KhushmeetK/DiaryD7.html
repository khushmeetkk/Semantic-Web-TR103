Day 7 - Web Identifiers and Web History

Learnings

On seventh day , we focused on key concepts in web standards, particularly how web addresses and resource identifiers are structured. We also explored the evolution of the web through its various stages:

  • URL (Uniform Resource Locator): The most common way of accessing resources on the web, specifying the location of a resource on the internet (e.g., a web page or file).
  • URN (Uniform Resource Name): An identifier that uniquely names a resource without providing its location (e.g., ISBN numbers for books).
  • URI (Uniform Resource Identifier): A general term encompassing both URLs and URNs, used to identify resources in a uniform manner across the web.
  • IRI (Internationalized Resource Identifier): An extension of URIs that allows the use of a wider range of characters from various languages, enabling more globalized web addresses.

Representation of RDF

We explored the common formats used to represent RDF (Resource Description Framework) data:

  • Turtle: A compact and human-readable format for expressing RDF triples, where subjects, predicates, and objects are written in a concise, structured manner.
  • RDF/XML: An XML-based format for encoding RDF data, often used for integrating RDF with other XML-based systems.
  • JSON-LD: A JSON-based format for representing linked data, making RDF easier to work with in web applications by leveraging JavaScript's JSON format.

History of the Web

The evolution of the web was covered through three major stages:

  • Web 1.0: The early "static web," where websites were largely read-only, offering basic text and images with little user interaction or content contribution.
  • Web 2.0: The rise of social media, blogs, and user-generated content. Websites became more interactive, with users contributing and sharing information across platforms.
  • Web 3.0: The "semantic web," aiming to make web data machine-readable and connected. Web 3.0 emphasizes linked data, AI-driven content, and a more personalized, decentralized web experience.
