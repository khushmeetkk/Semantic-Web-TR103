Day 1 - Introduction

Semantic Web

The Semantic Web is a vision of an extension of the existing World Wide Web, which provides software programs with a machine-interpretable method for data from the published information. In this, we add further data descriptions to the existing content and data on the web.

Importance

The Semantic Web improves the current web by allowing machines to understand and interpret the meaning of the data on web pages. It enables better data integration, automation, and interpretation across various systems, making it easier to find, share, and combine information. The need for the Semantic Web comes from the fact that the traditional web is mainly designed for humans, which limits how software can process and understand large amounts of unstructured data. By adding semantic meaning to data, the web becomes more accessible for machines, leading to smarter applications, more efficient data searches, and improved user experiences.

Understanding HTML & CSS

We took a deep dive into HTML and CSS—the foundational building blocks of web pages. HTML structures the content, while CSS styles it to make it visually appealing. Together, they create the user interfaces we interact with daily on websites. Today's practical session gave us hands-on experience in using these technologies to build simple yet functional elements of a webpage.
