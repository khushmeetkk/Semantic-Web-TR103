Day 2 - CSS Exploration

Learning About CSS

On the second day of training, we explored the features of Cascading Style Sheets (CSS). CSS plays a vital role in web development by defining how HTML elements appear visually on a webpage. It provides us with the tools to manipulate layout, colors, fonts, and overall design, enabling a more engaging user experience. By separating content (HTML) from presentation (CSS), we gain better control over our designs and can make updates easily. We also learned how CSS enhances the responsiveness of web pages, ensuring that they look good on various devices. Types of CSS

We explored three primary types of CSS:

  • Inline CSS: This type of CSS is applied directly within an HTML element using the style attribute. It is useful for quick styling changes but is not recommended for larger projects as it can clutter the HTML.
  • Internal CSS: This type is written within a <style> tag in the head section of an HTML document. It is suitable for styling a single page and keeps the styles in one place.
  • External CSS: This type is stored in a separate CSS file linked to the HTML document using the <link> tag. This method promotes reusability and keeps the HTML clean.

Box Models

The CSS box model is a fundamental concept that describes the rectangular boxes generated for elements in the document tree. It consists of several layers:

  • Content: The actual content of the box, such as text or images.
  • Padding: The space between the content and the border. Padding creates space inside the box, enhancing readability.
  • Border: The boundary that surrounds the padding (if any) and the content.
  • Margin: The space outside the border that separates the element from other elements on the page.

Containers and Div Element

We learned about containers and div element. A container is a structural element that holds content together, making it easier to apply styles and manage layouts. The <div> element is commonly used for grouping sections of a webpage. Using classes within the <div> tag allows for specific styling of groups of elements, making the webpage visually organized.
